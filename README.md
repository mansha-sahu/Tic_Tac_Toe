# Tic_Tac_Toe
-	Tic Tac toe is a simple and classic two-player game.
-	Built using HTML, CSS, and JavaScript.
- Play as Player O against Player X.
 
# Tech Stack
-	HTML5 – for structure
-	CSS3 – for styling
-	JavaScript – for game logic & AI
-	Minimax Algorithm – for decision making
